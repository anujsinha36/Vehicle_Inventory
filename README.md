# Vehicle Inventory

Vehicle Inventory is a modern Android application built using Kotlin and Jetpack Compose for managing and tracking vehicle records through a clean and intuitive UI.

The app allows users to add vehicles, manage inventory, view vehicle statistics, and apply filters based on brand and fuel type.

## Features

- Add vehicle details
- Vehicle inventory dashboard
- Brand & model selection flow
- Fuel type selection
- Filter vehicles by:
  - Brand
  - Fuel Type
- Vehicle statistics overview
- Modern UI built completely with Jetpack Compose
- Clean and responsive design

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Material 3
- Coroutines
